# neueduSpider
Neuedu spider

这是一个Neuedu的爬虫，可以抓取页面视频的直链并导入到其他播放器播放。
东软的视频网页播放器非常难用，如果浏览器不支持无法小窗播放，缩小屏幕视频分辨率极差，再加上东软的用户登陆维持时间非常短，经常看完一个视频又要重新登陆，这可能就是这个爬虫诞生的原因。
爬虫会自动把直链信息和标题关联起来并且可以导入到一些专业的本地视频播放器，这样就不需要反复挂着浏览器看视频了
